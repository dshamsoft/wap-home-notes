# wap-home-notes

## 简介

`wap-home-notes` 是一个用于归档和发布多个独立 HTML 页面的仓库。本仓库不针对任何特定域名或网站，仅作为静态页面集合的存储与展示空间。

## 目录说明

```
├── index.html          # 仓库主页面（可选）
├── pages/              # 存放独立 HTML 页面
│   ├── example-1.html
│   ├── example-2.html
│   └── ...
├── assets/             # 公共资源（样式、脚本、图片等）
│   ├── css/
│   ├── js/
│   └── images/
└── README.md           # 本文件
```

- `pages/`：存放所有独立 HTML 页面，每个文件均可直接通过浏览器打开或部署后访问。
- `assets/`：存放页面共享的静态资源，如样式表、JavaScript 文件、图片等。

## 页面归档说明

本仓库中的 HTML 页面均为独立归档，彼此之间无强关联。每个页面可单独使用、修改或删除。建议在新增页面时遵循以下规范：

- 页面文件名使用英文小写及连字符（如 `my-note.html`）
- 页面内引用资源使用相对路径，指向 `assets/` 目录
- 如有特殊依赖或说明，可在页面头部注释中注明

## 维护说明

- 本仓库由维护者不定期更新，新增或调整页面内容。
- 提交信息应简洁描述变更内容，如“新增页面：xxx”或“更新页面：xxx”。
- 欢迎通过 Issue 或 Pull Request 提出改进建议，但请注意保持仓库的通用性与简洁性。
